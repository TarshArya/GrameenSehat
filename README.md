# GrameenSehat – AI-Powered Telemedicine Platform

GrameenSehat is a web-based telemedicine application designed to improve healthcare accessibility in rural areas by providing quick preliminary health insights.

## 🚀 Features
- 🧠 AI-based symptom checker for basic health analysis  
- 📱 Simple and intuitive UI for non-technical users  
- 🌐 Responsive design for mobile and desktop  
- 🎨 Designed using a 14-page Figma prototype  

## 🛠️ Tech Stack
- React.js  
- TypeScript  
- Tailwind CSS  
- OpenAI API  

## 💡 Problem Statement
Access to healthcare in rural areas is limited. This project aims to provide a first-level digital consultation experience using AI to guide users based on symptoms.
